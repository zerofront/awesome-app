# awesome-app

提供IOS平台的限免资讯应用，用于实践 typescript + react + pwa + ssr

## run
- 安装依赖 `npm install`
- 启动 `npm run start`

## 相关
本项目仅供学习与交流使用。UI都是网络资源的。如有侵权，请联系wuqinghao@outlook.com
